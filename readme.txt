WPForce Logout

== Description ==

WPForce logout allows you (admins) to forcefully logout user(s).

Contribute on github: [GitHub repository](https://github.com/sanzeeb3/wp-force-logout)

### Features And Options:

* Logout specific user(s)
* Bulk logout users
* Users online/offline status display
* Last login activity
* Well Documented
* Translation ready
